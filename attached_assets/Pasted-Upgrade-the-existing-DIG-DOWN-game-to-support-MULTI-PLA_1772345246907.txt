INPUT SYSTEM ARCHITECTURE
--------------------------------------------------

Create an InputManager system that abstracts all inputs.

InputManager should expose normalized actions:

- moveLeft (0–1)
- moveRight (0–1)
- dashPressed (boolean)
- pausePressed (boolean)

PlayerController must read ONLY from InputManager, not directly from keyboard or touch events.

--------------------------------------------------
KEYBOARD SUPPORT (KEEP EXISTING)
--------------------------------------------------

- A / Left Arrow → move left

Refactor keyboard input into InputManager.

GAME CONTROLLER SUPPORT
--------------------------------------------------

- Support Xbox and PlayStation layouts.

Implementation details:
- Poll navigator.getGamepads().
- Apply deadzone (0.15) to analog stick.
- Smooth analog input for gradual movement speed.
